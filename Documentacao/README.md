# Documentação do leitor

- `YPD-R200 Product Datasheet V0.2.pdf`: datasheet do módulo especificado para o protótipo.
- `YPD-R200 User Protocol V2.3.3.pdf`: protocolo UART e conjunto de comandos.

Não confundir o módulo YPD-R200 com o leitor HARTING Ha-VIS RF-R200. Apesar do
nome semelhante, a documentação do equipamento HARTING não é fonte técnica para
o módulo de referência deste trabalho.
